## 1. Scope of Work

Contractor agrees to perform website development services for Client as described in the Atlas Analytics Website Upgrade Proposal (Version 2.0, dated March 15, 2026), which is incorporated by reference into this Agreement.

The work is organized into four phases:

**Phase 1 — Core Refresh & New Pages (Weeks 1-2)**
Elementor translation of fully coded Team, Careers, Testimonials pages. Global navigation and footer template setup. Yoast SEO configuration for meta descriptions across all pages.

**Phase 2 — Homepage & SEO Hardening (Weeks 3-4)**
Elementor translation of Homepage and Blog/Insights pages. Yoast SEO keyword implementation, structured data/schema markup, performance and accessibility audit.

**Phase 3 — Client Portal Foundation & Advanced Features (Weeks 5-7)**
Portal architecture planning and optional MVP build. Blog template Elementor translation. Job posting template system with ATS integration hooks.

**Phase 4 — Client Data API & Developer Portal (Weeks 8-12)**
RESTful API design and development for GDP nowcasts, trade indicators, and component forecasts. Authentication system, rate limiting, webhook support, developer documentation portal, WordPress integration layer, and security audit.

> **Phase Authorization:** Each phase requires written authorization from Client before work begins. Client is not obligated to authorize all phases. Phases 1-3 may proceed independently of Phase 4.
